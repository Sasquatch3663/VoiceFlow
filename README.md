# VoiceFlow
A smooth speech and text conversion tool. 
A simple Python program that converts speech to text and text to speech, using Google Speech Recognition & gTTS (Google Text-to-Speech).
Speech-to-Text ‚Äì Converts spoken words into text using Google Speech Recognition.
Text-to-Speech ‚Äì Converts any text input into speech using gTTS (Google Text-to-Speech).
Multi-language Support ‚Äì Supports multiple languages for text-to-speech conversion.
Interactive CLI ‚Äì User-friendly menu for easy navigation.
# Install Dependencies
pip install speechrecognition gtts playsound
# Usage
Choose an option:
1. Convert Speech to Text
2. Convert Text to Speech
3. Exit
